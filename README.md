# Too Yumm Chips Landing Page

A visually appealing and responsive landing page inspired by the **Too Yumm Chips** brand. This project is built entirely with **HTML5** and **CSS3**, focusing on modern UI design, product presentation, and clean layout structure.

## 🚀 Features

* Modern and attractive landing page design
* Responsive navigation bar with search section
* Hero section with product highlights
* Product showcase cards with ratings and pricing
* Custom feature icons and promotional content
* Clean and organized HTML structure
* Fully built using HTML and CSS (No JavaScript)

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Remix Icons
